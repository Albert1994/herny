## Ключевые концепции

### CIA Triad
- **Confidentiality** — доступ только authorized
- **Integrity** — данные не изменены неавторизованно
- **Availability** — доступность для authorized users

### Authentication & Authorization
- **Authentication** — who are you (OAuth 2.0, OIDC, SAML)
- **Authorization** — what can you do (RBAC, ABAC, ReBAC)
- **Zero Trust** — never trust, always verify
- JWT — structure, validation, pitfalls (don't store secrets in JWT)

### Security patterns
- **Defense in depth** — multiple layers
- **Least privilege**
- **Fail secure** — deny by default
- **Secrets management** — Vault, AWS Secrets Manager
- **Encryption**: at rest (AES), in transit (TLS 1.3)
- **Input validation** — OWASP Top 10

### OWASP Top 10 (awareness)
- Broken Access Control, Injection, SSRF
- Security Misconfiguration
- Vulnerable Components

### Compliance
- GDPR — data privacy, right to erasure
- PCI DSS — payment card data
- SOC 2 — trust services criteria
- Data residency requirements

### Threat modeling
- STRIDE: Spoofing, Tampering, Repudiation, Info Disclosure, DoS, Elevation
- Attack trees
- When in SDLC: design phase
