## Проект этапа: CRUD-сервис с доменной моделью

### Задача

Реализовать сервис управления задачами (Task Manager) или заказами (Order Service):

- REST API с валидацией
- Доменная модель с инвариантами (не анemic model)
- PostgreSQL + миграции
- Unit-тесты на доменную логику, integration-тесты на API
- README с описанием модели и API

### Критерии сдачи

- [ ] Покрытие доменной логики тестами ≥ 80%
- [ ] SOLID нарушения можно аргументированно объяснить (или исправлены)
- [ ] ER-диаграмма и описание агрегатов в README
